Here are some of things I'm working on:

IdioMatch is a language matchmaking service that pairs users depending upon which language they are trying to learn! Six languages are currently supported. Built with Node.js, Express, HTML5, CSS, Foundation, PostgreSQL, and Passport.js for user authentication.

A mock forum built for students at General Assembly. Users can ask questions, respond with comments and navigate to helpful documentation by clicking on the provided links. Built with: HTML,CSS, Express, React.js, PostgreSQL and Bootstrap.

A spanish verb conjugator that outputs the conjugated form of four tenses in the indicative mood. Built with Node.js, Express, PostgreSQL, HTML, and CSS.
